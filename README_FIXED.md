# PGK Construction Website (Fixed)

This project has been updated with bug fixes, UI improvements, and responsive design.

## How to Run

1.  **Install Dependencies**:
    Open a terminal in this folder and run:
    ```bash
    npm install
    ```

2.  **Start the Server**:
    Run the following command to start the development server:
    ```bash
    npm start
    ```
    The website will open in your browser at `http://localhost:3000`.

## Changes Included
- Fixed `ConstructionContext` errors.
- Added background image to Construction page.
- Fully responsive design for mobile and tablet.
- Improved UI aesthetics.
- Organized assets.

## Build for Production
To create a production build, run:
```bash
npm run build
```
This will create a `build` folder that can be deployed to any static hosting service.
